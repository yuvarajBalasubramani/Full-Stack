└────────────────────────────────────────────────────────────────────────────────┘

  STEP 1: Get Google Maps API Key
  ─────────────────────────────────
    → Go to: https://console.cloud.google.com/
    → Enable: Maps JavaScript API, Geocoding API, Directions API
    → Create API Key

  STEP 2: Configure .env File
  ────────────────────────────
    → Open: .env
    → Add: VITE_GOOGLE_MAPS_API_KEY=your_key_here

  STEP 3: Start Servers
  ──────────────────────
    Terminal 1: cd server && npm run dev
    Terminal 2: npm run dev

  STEP 4: Test!
  ──────────────
    → Open: http://localhost:5174
    → Login and add items to cart
    → Test checkout with different payment methods
    → Track your order on Google Maps
    → Try cancelling an order

  PAYMENT FLOW:
    1. Add items to cart
    2. Click "Proceed to Checkout"
    3. Fill shipping address
    4. Select payment method:
       • Google Pay (instant payment)
       • Cash on Delivery (pay later)
       • Card (enter card details)
       • UPI (enter UPI ID)
    5. Complete order
    6. See success screen with order details

  ORDER TRACKING:
    1. Click Package icon (📦) in navigation
    2. View all your orders
    3. Click "Track Order" on any order
    4. See Google Maps with:
       • Delivery route
       • Delivery person location
       • Distance calculation
       • Delivery person info
       • Status timeline

  ORDER CANCELLATION:
    1. Place a new order
    2. Go to order tracking page
    3. See yellow banner "You can cancel this order"
    4. Click "Cancel Order"
    5. Enter reason and confirm
    6. See refund initiated (for online payments)

     → Make sure backend server is running on port 5000

  ❌ Cancellation not working?
     → Only orders within 1 hour can be cancelled
